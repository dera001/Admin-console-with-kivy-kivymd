# Admin-console-with-kivy-kivymd

# Requirements
* Python3  [ python3 >= 3.6]
* kivy 
* kivymd [ kivymd == 0.104.0dev ]

# installation
```sh
pip install kivy[base] kivymd
```
# usage
* clone the repo
* cd into the main project directory
* run this command on the terminal

```sh
python3 main.py
```
